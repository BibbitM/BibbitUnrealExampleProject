# Bibbit Unreal Example Project

A project for testing and developing the Bibbit Unreal Utilities plugins.

To develop the plugin:
- Clone the repository [BibbitUnrealUtilities](https://github.com/BibbitM/BibbitUnrealUtilities).
- Link the local folder to \Plugins\Bibbit with commands:
  - `mkdir Plugins`
  - `cd Plugins`
  - `mklink /J Bibbit ..\..\BibbitUnrealUtilities`
